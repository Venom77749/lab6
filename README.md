### Задание 2. Dictionary
На вход программы подаются фамилии и имена учеников. Известно, что общее количество
учеников не превосходит 100. В первой строке вводится количество учеников, принимавших
участие в соревнованиях, N. Далее следуют N строк, имеющих следующий формат:

  <Фамилия><Имя>
  
Здесь <Фамилия> – строка, состоящая не более чем из 20 символов; <Имя>– строка, состоящая не
более чем из 15 символов. При этом <Фамилия> и <Имя> разделены одним пробелом. Примеры
входных строк:

- Иванова Мария
- Петров Сергей

Требуется написать программу, которая формирует и печатает уникальный логин для каждого
ученика по следующему правилу: если фамилия встречается первый раз, то логин – это данная
фамилия, если фамилия встречается второй раз, то логин – это фамилия, в конец которой
приписывается число 2 и т.д. Например, для входной последовательности:

- Иванова Мария
- Петров Сергей
- Бойцова Екатерина
- Петров Иван
- Иванова Наташа

будут сформированы следующие логины:

- Иванова
- Петров
- Бойцова
- Петров2
- Иванова2

### Алгоритм решения
1. Хранить динамический массив структур Node, где каждый Node содержит key и value.  
2. Поддерживать переменные size (текущее число элементов) и capacity (ёмкость массива).  
3. При добавлении нового элемента:
   - если массив заполнен (size == capacity), удвоить capacity, создать новый массив и скопировать элементы.  
4. Для поиска индекса по ключу проходить массив от 0 до size - 1 и сравнивать data[i].key с заданным ключом.  
5. Вставка (insert):
   - найти индекс по ключу;  
   - если ключ существует, заменить value;  
   - если нет, при необходимости расширить массив и добавить новый Node в позицию size, увеличить size.  
6. Получение (get):
   - найти индекс по ключу;  
   - если найден, вернуть value;  
   - иначе вернуть значение по умолчанию (ValueType()).  
7. Удаление (remove):
   - найти индекс по ключу;  
   - если не найден, вернуть false;  
   - если найден, сдвинуть все элементы правее на одну позицию влево, уменьшить size, вернуть true.  
8. Печать (printAll):
   - если size == 0, вывести «Словарь пуст!»;  
   - иначе для каждого элемента вывести key и value.

формирование логинов

1. Открыть файл students.txt для чтения.  
2. Считать число N — количество учеников; проверить, что 0 ≤ N ≤ 100.  
3. Создать словарь Dictionary<std::string, int> surnameCount.  
4. Для каждого из N учеников:
   - считать surname и name;  
   - проверить, что длина фамилии не больше 20, а имени — не больше 15 символов;  
   - запросить из словаря значение count = surnameCount.get(surname) (сколько раз фамилия уже встречалась; если первый раз, будет 0);  
   - если count == 0, логин = surname;  
   - иначе логин = surname + std::to_string(count + 1);  
   - вывести логин;  
   - записать обратно в словарь surnameCount.insert(surname, count + 1) для обновления счётчика.

### Задание 3. Наследование
Автомат. Создайте такой подвид сущности Пистолет из задачи 3.5.1, которая будет совпадать с
ней во всех отношениях, кроме следующего:
- Имеет скорострельность (целое число) которое обозначает количество выстрелов в
секунду, поддерживаемое данным автоматом. Скорострельность всегда положительное
число.
- При вызове Стрелять количество выстрелов соответствует скорострельности (например,
при скорострельности 3 выводится три строки с текстом выстрела).
- Умеет Стрелять N секунд, что приводит к количеству выстрелов равному N умноженное
на скорострельность.
- Инициализация может быть выполнены следующими способами:
  
a) Без параметров. Скорострельность 30, вместимость 30.

b) С указанием максимального числа патронов. Скорострельность будет равна
половине обоймы

c) С указанием максимального количества патронов в обойме и скорострельности

### Алгоритм решения
Пистолет 

1. Объявить класс Pistol с полем ammo (количество патронов).
2. В конструкторе принять начальное число патронов и записать его в ammo.
3. При вызове метода shoot():
- если ammo > 0, вывести «Бах!», уменьшить ammo на 1;
- иначе вывести «Клац!».
4. В main:
- запросить у пользователя количество патронов и число выстрелов;
- создать объект Pistol;
- в цикле shots раз вызывать shoot().

Автомат

1. Объявить класс Automatic, унаследованный от Pistol..
2. Добавить поля rateOfFire (скорострельность) и maxAmmo.
3. Реализовать три конструктора:
- без параметров: ammo = 30, rateOfFire = 30, maxAmmo = 30;
- с параметром maxCapacity: ammo = maxCapacity, rateOfFire = maxCapacity / 2, maxAmmo = maxCapacity;
- с параметрами maxCapacity и fireRate: ammo = maxCapacity, rateOfFire = fireRate, maxAmmo = maxCapacity.
4. В методе shoot():
- выполнить цикл от 0 до rateOfFire - 1, при этом пока ammo > 0 печатать «Бах!» и уменьшать ammo;
- если после цикла ammo == 0, вывести «Клац!».
5. В методе shootForSeconds(seconds):
- вычислить shots = seconds * rateOfFire;
- выполнить цикл shots раз, при этом пока ammo > 0 печатать «Бах!» и уменьшать ammo;
- если патроны закончились, вывести «Клац!».
